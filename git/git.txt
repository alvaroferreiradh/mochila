COMANDO DO GIT
 - git init -> Inicia um repositório
 - git add . -> Adiciona todos os arquivos no repositório
 - git commit -m "comentario do commit" -> Faz um commit
 - git checkout -b nomeDaBranch -> Troca de branch e cria, caso não exista.
 - git merge nomeDabranch -> Junta o branch com a atual.
 - git remote -> Trata de repositórios em núvem
 - git push origin main -> Empurrar nossa branch para o Github


PROCESSOS DO DIA A DIA
 - git status (status do repositorio)
 - git add . (adiciona/prepara os arquivos)
 - git commit -m "descricao do commit" (cria um commit)
 - git push (sobe para o github)
